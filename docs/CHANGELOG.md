# Changelog

All notable changes to TrackMySpend will be documented in this file.

## [1.0.0] - 2025-11-01

### Added
- Initial release of TrackMySpend personal finance manager
- User authentication with secure password hashing
- Transaction management with categories
- Budget planning and tracking
- Smart financial insights and recommendations
- Calendar view for financial activities
- Bill reminders and recurring payments
- Responsive design for desktop and mobile
- Comprehensive deployment guide

### Security
- bcrypt password hashing implementation
- JWT token-based authentication
- Environment-based configuration
- Input validation for user registration and login

### Developer Experience
- One-command startup for both frontend and backend
- Comprehensive README with setup instructions
- Environment configuration templates
- Production deployment documentation
